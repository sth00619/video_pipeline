# Image style bible

Every generated scene uses one locked, original visual language: 2D Korean webtoon illustration, dark ink outlines, cel shading, saturated controlled color, and a dense topic-specific background. Photorealistic, 3D, and mixed-media compositing are prohibited.

`Goldie` is the project's own gold-coin mascot. Do not reproduce another channel's green banknote character, logo, or proprietary design. Goldie changes role, costume, action, expression, and relevant props for each scene, while its face and silhouette remain fixed by the approved reference sheet.

Generated images contain no typography. Headlines are rendered deterministically after generation; narration subtitles remain timed ASS overlays during final assembly. Each image request is 16:9 and carries the same style lock plus up to two approved reference sheets.
